# Huddle Landing Page

This is my solution to the **Huddle Landing Page challenge** from Frontend Mentor.

Live Demo: https://huddle-landing-page-omega-two.vercel.app/  
GitHub Repo: https://github.com/Ta-xr/Huddle-Landing-Page

---

## 📌 Overview

This project is a responsive landing page built using **HTML and CSS only**.  
It focuses on recreating a modern UI design from Frontend Mentor as accurately as possible.

---

## 🎯 The Challenge

Users should be able to:

- View the optimal layout depending on their device’s screen size
- See hover states for interactive elements

---

## 🛠️ Built With

- HTML5 semantic markup
- CSS3
- Flexbox
- Responsive Design
- Mobile-first workflow

---

## 💡 What I Learned

During this project, I improved my skills in:

- Structuring clean and semantic HTML
- Building responsive layouts using Flexbox
- Handling spacing, alignment, and typography
- Translating a UI design into real code
- Improving attention to detail in frontend development
